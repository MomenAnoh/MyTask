# Task Management Application

## Project Description
This is a simple Task Management application built with **Laravel** and **Blade** for the frontend.  
It allows users to register, log in, and manage their tasks (create, update, delete, and change status).

---

## My Experience
- **Main Expertise:** Backend development with Laravel.  
- **Frontend:** I used Blade, HTML, and CSS, improving the interface and functionality with the help of AI tools.  
- **Laravel Breeze:** Used to simplify authentication setup (registration and login).

---

## Technologies Used
- **Backend:** Laravel 12, PHP 8.2  
- **Frontend:** Blade, Bootstrap, CSS (main.css)  
- **Database:** MySQL (SQLite can also be used)  
- **Authentication:** Laravel Breeze (session-based)  

---

## Project Features
- User registration and login  
- Create, update, and manage tasks  
- Display tasks in a styled table using Bootstrap  
- Update task status (Pending, In Progress, Done)  
- Delete tasks  
- Each user can only see their own tasks  
- Clean and readable UI
